ENHACING CONTENT PRIVACY IN SOCIAL NETWORK USING DIGITAL RIGHT MANAGEMENT
Topic

The aim of this project is to demonstrate how digital copyrighted content can be protected in a social media network

This solution would be make use of Three (3) approach in securing copyrighted content
1. Identify if uploaded content is copyrighted
The first thing social media network should enforce is to identify if any content being uploaded is copyrighted.
This solution would provide such features. This would be achieved through reading images, documents metadata.

2. Preventing content from being downloaded or copied
Social media networks are designed to display content from users.
In the case a copyrighted file is being displayed, such is not allowded to downloaded.
